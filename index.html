AgentsDash.com

Realestate Agents Performance Dashboard

New

I developed this web application from scratch using Django Framework. I used Jquery, a javascript library for DOM manipulation and asynchronous API calls. I was solely responsible for all aspects of this project i.e. Design, Development and Deployment. I used Git as version control tool throughout. I used dbsqlite for development and MySQL database for production. I deployed it on DigitalOcean with uwsgi and nginx. For tabular representation and efficient analysis of data, I customised DataTables. I also customised HightCharts, a javascript graph library for graphical representation and prediction of future sales.
